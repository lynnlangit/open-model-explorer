Upload limits
Images: up to 3 files, ≤ 10 MB each.
Formats: PNG, JPG, WEBP, GIF, BMP, TIFF.
Text prompt: up to 4,000 characters.
Note: images are re-encoded (lossless PNG, ≤ 2048 px) for the model. Vision models ingest 8-bit tiles, so this is a research/exploration tool — not diagnostic-grade viewing of 16-bit or whole-slide scans.

Add a model

Enter a model ID (not its display name) in one of two forms, then Download. It pulls to this Mac and appears in the picker.

  • Ollama library name — e.g. mistral:7b, medgemma, llava:7b
  • Hugging Face GGUF repo — e.g. hf.co/username/Repo-Name-GGUF
